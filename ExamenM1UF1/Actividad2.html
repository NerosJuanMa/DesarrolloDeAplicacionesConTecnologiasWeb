Lenguaje de Marca: Estilos en CSS

Tu misión: aplicar 6 formas de dar estilo, siguiendo las instrucciones.

1) Selector universal

Este selector aplica estilos globales como márgenes y el box-sizing.

2) Selector de tipo

Este texto se estiliza usando el selector de tipo para los párrafos.

3) Clase

Este párrafo usa una clase para resaltar su estilo.

4) ID

El encabezado de esta página tiene un ID que se usa para aplicarle un estilo único.

5) Cambiar la tipografía

Se aplica una fuente tipográfica globalmente mediante la propiedad font-family.
